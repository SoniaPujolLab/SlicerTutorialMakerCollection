## <span style="color: #003366; font-size: 3.5rem; font-weight: 600; display: block; margin-bottom: 30px; padding-bottom: 15px; border-bottom: 3px solid #003366;">Básico de Slicer5</span>

<div style="font-size: 2rem; line-height: 1.8; color: #444; text-align: justify;">

<p>*Slicer es un software de código abierto para la segmentación, el registro y la visualización de datos de imágenes médicas.</p><p>*La plataforma se desarrolla mediante un esfuerzo conjunto de varias instituciones, incluyendo consorcios a gran escala financiados por los NIH.</p><p>*Slicer es exclusivamente para investigación médica y no cuenta con la aprobación de la FDA. </p>

</div>

<div style="font-size: 2rem; line-height: 1.8; color: #444; text-align: justify;">
